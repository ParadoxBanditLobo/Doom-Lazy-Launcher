# Doom Lazy Launcher vX.X

Tiny terminal launcher for Doom source ports on Linux.

## Download

Use the Linux x86_64 executable from this release.

## Run

```sh
chmod +x DoomLazyLauncher-vX.X-Linux-x86_64
./DoomLazyLauncher-vX.X-Linux-x86_64
```

## Notes

- No third-party libraries are required.
- Does not include Doom, Doom engine files, IWADs, mods, or copyrighted game data.
- You must provide your own Doom source port and legally obtained IWADs.
